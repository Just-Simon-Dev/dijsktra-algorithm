Welcome to Dijkstra Algorithm Helper

Here I will explain a bit how to work with this visualization

First icons:

means that you can add a new start block to the grid, you can only have one start block

means that you can add a new end block to the grid, you can only have one end block

means that you can add a new wall block to the grid

means that you can remove a block from the grid

means that you can add to the block a weight, just click on the block and a prompt will appear asking for the weight

means that you reset all grid except start and end blocks

Remember to click first the icon that you want to use and then click on the block that you want to change.

After you have set up the grid, you can click on the button "Use Dijkstra Algorithm" to start the algorithm.

The gray blocks that are appearing are the ones that was visited by the algorithm and black ones are the path.
